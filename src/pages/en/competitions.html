Competitions

If any of the platforms has less than 3 entries in any compo, we are going to join demos and intros together and, if this is not enough, use the former oldschool / newschool formula. We will also be flexible with other compos if they are very popular (separating different kinds of tracker formats if needed, etc.), all to make the competitions more exciting. As this is a multiplatform party, any demoscene hardware is welcome.

Rules

“Space and Flying Saucers” is our theme this year. We encourage you to include the theme in your prod. The best and most clever uses will be rewarded with a sur-prize!

General rules

  • Remote entries are allowed, see How to submit your entry
  • Only new entries, not submitted anywhere else
  • No copy / no timbaland / no moroder
  • We reserve the right to reject/remove any entries which do not follow the rules or are not demoscene related
  • If your prod violates any Polish or international law, is considered offensive or inappropriate, we can disqualify it or exclude it from post-party submission to demoscene archives

Freestyle GFX

Pictures made with any technique.
  • Any resolution (our beamer is 1920x1080 max)
  • 3D renders, shopped pictures and all which does not qualify for the Pixel GFX category
  • Popular file format (IrfanView will be used)
  • Please submit work stages

Pixel GFX

Pixel-art for any platform.
  • Resolution: 320 x 256 max, 256 colours max
  • Please submit work stages
  • In case of Atari/C-64/ZX Spectrum entries please submit an executable file (*.xex, *.prg, *d64 or*.tap), can also be a 5.25” floppy or *.d64 files (C-64).

Streaming MSX

Music created with any technique.
  • Allowed formats: MP3/Ogg Vorbis
  • Max playtime — 3 minutes (entries can be longer but only the initial 3 minutes will be played at the party)

Tracked MSX

Music made in trackers, samples allowed.
  • Allowed formats: MOD (Sound/Noise/Fast/ProTracker), XM (Fasttracker II), S3M (ScreamTracker 3), IT (ImpulseTracker), DIGI (DigiBooster), AON (Art Of Noise), MED (OctaMED), FC (Future Composer), AHX/THX (AHX), HVL (HivelyTracker), DMF (DefleMask), FTM (FamiTracker)
  • Format list is open. If you want to submit entry in some other format, please contact us.
  • Max filesize 2 MB (2097152 bytes)
  • Don't use any packer/cruncher (MO3 etc.).
  • We don't allow using formats that use advanced, internal effects (RNS, XRNS, DBM, SKM, MT2, KT)
  • Also, your file can't use external plugins and VST (tune will be played on clean installation of chosen tracker).
  • To avoid any doubts, please specify the exact name and version of the tracker. In the absence of such information, we will try to use the tracker for the native format (MOD - ProTracker, XM - Fast Tracker II, etc.)
  • Please also provide an MP3 version of the tune (for the lazy listeners :))
  • Max playtime — 3 minutes (entries can be longer but only the initial 3 minutes will be played at the party)

Chip MSX

Chiptune music made using only specialized sound chips.
Any synthesis based formats (please provide us with either an executable or a regular file) which utilise any of the following sound chips:
  • SID (6581/6582/8580), VIC, TED chips (Commodore)
  • POKEY, TIA (Atari)
  • PSG chips: YM2149, AY-3-89xx, SN764xx (Atari ST, ZX Spectrum, Amstrad CPC, MSX, Sega Game Gear, Sega Master System, Sega Mega Drive)
  • Philips SAA1099 (SAM Coupé, Game Blaster, Silicon Graphics ;) )
  • Yamaha FM chips: OPL, OPN, OPM families (AdLib, Turbosound FM, Sega Master System, Sega Mega Drive, fancy Japanese computers/arcades ;) )
  • Game Boy products family (GB, GBC, GBA)
  • DAVE (Enterprise 64/128)
  • Ricoh 2A03/2A07, Ricoh 2C33 family (NES, Famicom Disk System)
  • Konami RC, VRC6, VRC7, Nintendo MMC5, Namco 163 (NES cartridges)
  • Konami SCC/SCC+ (MSX cartridges)
  • Hudson Soft HuC6280 (PC Engine)
  • No samples allowed, only pure chip synthesis!
  • You may use more than 1 chip at once, for example: dual SID, 2 Game Boys playing simultaneously, etc.
  • The music will be played from the real hardware. Even though we will have most of popular machines (Commodore 64, Atari XE, etc.) present at the party place, please keep in mind we don’t own each and every bit of hardware or software capable of playing music for the aforementioned chips.
  • Max playtime — 3 minutes (entries can be longer but only the initial 3 minutes will be played at the party).
For the above reason, please:
  • Contact us beforehand to confirm if we’ll be able to play your file
  • Regardless of the above, please deliver additionally an MP3 recording of your tune, or even better, a video capture (preferred!)
  • Last, but not least: if you manage to find (and utilise) even more obscure hardware than listed above (like TI-82 Calculator), e-mail us to confirm that your entry will be allowed in this competition.

How to submit your entry

  • Entries at the party should be physically submitted to the compo team (pendrive, USB HDD, CD, floppy) before the deadline
  • Remote entries are allowed but you will not receive the statue as we are not going to send it via post. In case of a remote submission, please email us (contact@riverwash.org) an attachment or a link to a cloud service of your choice. The deadline for remote entries is 02.09.2016.
  • Only compressed archives (zip/7zip) with the following naming pattern: componame_nick/group_title.zip (newschoolgfx_george_landscape.zip)
  • Please include an info file with details in the archive
